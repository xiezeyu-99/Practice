#  LeetCode-319-bulb-switcher

>题目链接

[LeetCode-319-bulb-switcher](https://leetcode-cn.com/problems/bulb-switcher/)

>思路

完全平方数

>代码

```go

func bulbSwitch(n int) int {
    return int(math.Sqrt(float64(n)))
}


```
